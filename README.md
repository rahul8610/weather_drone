🛰️ Weather Prediction System for Drones using ROS 2 and Machine Learning

This project implements an AI-powered weather forecasting system for drones. It uses real-time telemetry from a simulated drone (via ArduPilot SITL + MAVROS + ROS 2) and machine learning trained on historical weather data to predict:

    🌧️ Rainfall
    ☁️ Cloud Cover
    🌅 Sunrise and Sunset Time
    🌡️ Temperature Range (Min/Max)
